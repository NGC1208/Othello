# CPU難易度選択

## 目的

対局開始前に初級・中級・上級を選択でき、カジュアルプレイヤーから経験者まで自分に合ったCPUと対局できるようにする。

## 作業範囲

- 起動直後の難易度選択ダイアログ
- 初級の初心者評価と1手先探索
- 中級の戦略評価と2手先探索
- 上級の戦略評価と3手先探索
- 対局中の難易度表示
- 再対局時の難易度再選択
- 難易度別の自動テスト

## 作業対象外

- 対局途中での難易度変更
- 確率だけで悪手を選択する処理
- 合法手からの無作為選択
- プレイヤー対プレイヤー
- オンライン対戦

## 受け入れ条件

- [x] 起動後、CPUが着手する前に難易度選択ダイアログが表示される。
- [x] 初級・中級・上級のいずれかを選択しない限り対局が始まらない。
- [x] 初級は初心者評価で1手先を評価する。
- [x] 中級は戦略評価で2手先まで探索する。
- [x] 上級は戦略評価で3手先まで探索する。
- [x] 選択中の難易度が対局画面へ表示される。
- [x] 「もう一度遊ぶ」で難易度を再選択できる。
- [x] 全難易度でCPUは必ず合法手を選択する。
- [x] 同じ盤面と難易度では同じ手を選択する。

## テスト計画

- 各難易度の探索深度を確認する。
- 初級が隅の隣接マスを常に回避しない局面を用意して確認する。
- 難易度選択前にCPUが着手しないことを確認する。
- 再対局時にダイアログが再表示されることを確認する。
