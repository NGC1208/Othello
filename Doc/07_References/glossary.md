# 用語集

仕様・コード・UIで同じ用語を使用するための一覧です。

| 英語 | 日本語 | 定義 |
|---|---|---|
| Board | 盤面 | 8×8のマス全体 |
| Cell | マス | 盤面上の1区画 |
| Disc | 石 | 黒または白の駒 |
| Legal move | 合法手 | ルール上、石を置ける手 |
| Flip | 反転 | 挟まれた相手の石を自分の色へ変えること |
| Pass | パス | 合法手がないため手番を相手へ渡すこと |
| Difficulty | 難易度 | CPUの評価方式と探索深度の組み合わせ |
| Beginner | 初級 | 初心者評価を使い、CPU自身の1着手だけを評価する難易度 |
| Intermediate | 中級 | 戦略評価を使い、CPUと人間の2着手を探索する難易度 |
| Advanced | 上級 | 戦略評価を使い、CPU・人間・CPUの3着手を探索する難易度 |
| Iterative deepening | 反復深化 | 深さ1から順に探索し、時間内に完了した最深結果を採用する方式 |
| Undecided | 未定 | 後で決定・記入が必要な項目 |
