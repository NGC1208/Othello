# ゲームモード

## モード一覧

| モード         | 説明  | 優先度 | 状況  |
| ----------- | --- | --: | --- |
| プレイヤー対CPU | 初級・中級・上級から難易度を選択して対局する | 必須 | 採用 |
| プレイヤー対プレイヤー | 人間同士の対局 | 対象外 | 不採用 |
| オンライン対戦 | ネットワークを使用した対局 | 対象外 | 不採用 |

## プレイヤー対CPU

- 先手・後手選択: 先手CPU
- CPU難易度: 対局開始前に初級・中級・上級から選択する。
- 難易度の変更: リザルト画面から再対局するときに再選択する。
