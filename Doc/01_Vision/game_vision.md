# ゲームビジョン

## ゲーム概要

コンピューター対人間のオセロゲームです。
プレイヤーは初級・中級・上級から難易度を選び、判断傾向と先読みの深さが異なるCPUと対局します。


## プレイヤー体験

- 初心者は、目先の石数を重視する初級CPUを相手に勝利を目指せる。
- ルールに慣れたプレイヤーは、中級・上級へ段階的に挑戦できる。
- 起動後すぐに難易度を選び、余計な画面操作なしで対局を始められる。

## 対象プレイヤー

- 想定プレイヤー: オセロ初心者から経験者
- プレイ経験: 初心者～中級者
- 1回の想定プレイ時間: 3～10分

## 製品範囲

- 対応プラットフォーム: Windows
- 販売形態: 未定
- 価格: 未定
- 対応言語: 日本語

## 対象外とするもの

- プロ並みの強さを持つCPU
- プレイヤー対プレイヤー
- オンライン対戦
- タイトル画面、モード選択画面、ポーズ画面、設定画面
- BGMおよび効果音
