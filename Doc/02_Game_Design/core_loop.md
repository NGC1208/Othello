# コアループ

## 基本の繰り返し

1. 局面を確認する。
2. 石を置く場所を選ぶ。
3. 石を置き、相手の石を裏返す。
4. 相手が手を指す。
5. 終局まで繰り返す。

## 1回のプレイの流れ

1. ゲームを起動する。
2. 難易度選択ダイアログで初級・中級・上級のいずれかを選ぶ。
3. CPUが黒、人間が白として対局を開始する。
4. 終局まで交互に着手する。
5. リザルトで勝敗と最終石数を確認する。
6. 「もう一度遊ぶ」で難易度を再選択するか、「ゲームを終了」で終了する。

## 繰り返し遊ぶ動機

<!-- 再対局したくなる理由 -->

- 異なる難易度へ挑戦し、CPUの判断傾向の違いを楽しむ。
- 同じ難易度へ再挑戦し、勝利を目指す。
