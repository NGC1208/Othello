# 完成条件

## 機能ごとの完成条件

- [ ] 承認済み仕様と一致している。
- [ ] 受け入れ条件をすべて満たしている。
- [ ] ビルドが成功する。
- [ ] 関連する自動テストが成功する。
- [ ] エンジン上で動作確認済みである。
- [ ] エラーや警告を不必要に増やしていない。
- [ ] 関連文書を更新している。
- [ ] 既知の制約を記録している。

## 製品版の完成条件

- [ ] 全ゲームモードを最後までプレイできる。
- [ ] 重大な既知不具合がない。
- [ ] 対象端末で性能予算を満たしている。
- [ ] 使用素材の商用ライセンスを確認済みである。
- [ ] ストア提出物を確認済みである。
