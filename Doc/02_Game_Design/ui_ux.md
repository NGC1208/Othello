# UI / UX仕様

## 画面一覧

- 対局画面
- リザルト表示

## 初期版の対象外

- タイトル画面
- モード選択画面
- ポーズ画面
- 設定画面

ゲームを起動すると、プレイヤー対CPUの対局画面を直接表示する。

## 対局画面

- 盤面表示: 8x8マスの2D表示で完全俯瞰
- 現在の手番表示: 「CPUの手番」または「あなたの手番」を常時表示する。
- 石数表示: 黒と白の石数を常時表示する。
- 合法手の表示: 表示しない。
- 最後の着手位置: 最後に石を置いたマスを枠で強調表示する。
- 対局中の戻る／やり直し: 初期版では実装しない。

## パス

- 人間に合法手がない場合、「あなたは置ける場所がないためパスします」と約1秒表示し、CPUの手番へ自動進行する。
- CPUに合法手がない場合、「CPUは置ける場所がないためパスします」と約1秒表示し、人間の手番へ自動進行する。
- 両者に合法手がない場合はリザルト表示へ移行する。

## リザルト表示

- 勝敗を「あなたの勝ち」「あなたの負け」「引き分け」のいずれかで表示する。
- 黒と白の最終石数を表示する。
- 「もう一度遊ぶ」ボタンで初期盤面から再対局する。
- 「ゲームを終了」ボタンでアプリケーションを終了する。

## 入力方法

- マウス: 合法な空きマスを左クリックすると石を配置し、挟まれた相手の石が反転する。
- タッチ: 初期版では対応しない。
- ゲームパッド: 初期版では対応しない。
- キーボード: 初期版では対局操作に使用しない。

## 操作への反応

- 石を置いた演出: 石を即座に配置する。
- 石を裏返す演出: 成立したすべての方向の石を、同時に約0.2秒で反転させる。
- 不正な入力への反応: 合法手ではないマスをクリックしても何も起こさない。
- 勝敗演出: リザルト表示を画面上に重ねて表示する。追加の演出は初期版では実装しない。

## アクセシビリティ

- 色覚への配慮: 未定
- 文字サイズ: 未定
- アニメーション軽減: 未定

## ウィンドウ

- 基準解像度: 1280 × 720
- 起動方式: ウィンドウ表示
- ウィンドウサイズ変更: 許可する。
- 画面比率: 16:9を基準とし、サイズ変更時も盤面の縦横比を維持する。
- 全画面表示: 初期版では実装しない。
