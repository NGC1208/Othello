# ビジュアル・オーディオ方針

## アート方針

- テーマ: 標準的で見やすいオセロ盤
- キーワード: シンプル、明瞭、装飾を抑える
- 参考作品: 特定作品の外観は模倣せず、一般的なオセロ盤を基準とする。
- 避ける表現: 盤面や石の状態を判別しにくくする過度な装飾

## ビジュアル仕様

- 基準解像度: 1280 × 720
- アスペクト比: 16:9
- 盤面デザイン: 緑色の8×8マス
- 石のデザイン: 黒と白の円形
- 背景: 暗い単色
- UIスタイル: 装飾の少ないシンプルなデザイン

## アニメーション

- 石の配置: 即座に表示する。
- 反転: 対象となる石をすべて同時に約0.2秒で反転する。
- 画面遷移: 起動時と再対局時に難易度選択ダイアログを重ね、終局時にリザルト表示を重ねる。

## 音響

- BGM方針: 初期版では使用しない。
- 効果音方針: 初期版では使用しない。
- 音量設定: 初期版では実装しない。

## 素材のライセンス

- 初期版は盤面、石、背景、UIをGodotの描画機能と標準UIで生成する。
- 外部画像・フォント・音源素材は使用しない。
- 将来、外部素材を追加する場合は出典、ライセンス、商用利用可否を記録する。
