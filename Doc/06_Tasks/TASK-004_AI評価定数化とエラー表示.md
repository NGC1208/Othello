# AI評価定数化と復旧不能エラー表示

## 目的

CPU評価関数の調整意図をコード上で明確にし、CPU処理から復旧できない場合にプレイヤーへ安全な案内を表示する。

## 実装内容

- 初級評価の反転数、石数差、角、辺、危険マスの重みを名前付き定数へ変更した。
- 戦略評価の局面境界、合法手、角、危険マス、石数差、終局スコアを名前付き定数へ変更した。
- CPUタスクが失敗・キャンセルされた場合は先頭合法手へ復旧する。
- CPUが非合法手を返した場合も先頭合法手へ復旧する。
- 合法手なしの場合は通常のパスとして処理する。
- 合法手が存在するにもかかわらず復旧できない場合は、エラー画面を表示して対局を安全停止する。

## 作業対象外

- CPUタスクのキャンセル機構
- 合法手計算の最適化
- 盤面複製方式の最適化

## 受け入れ条件

- [x] 評価式に直接記述されていた重みと局面境界が名前付き定数になっている。
- [x] 復旧成功時はゲーム内エラーを表示しない。
- [x] 通常のパスをエラーとして扱わない。
- [x] 復旧不能時はプレイヤーへエラーを表示し、対局を停止する。
- [x] 再対局またはゲーム終了を選択できる。
- [x] ビルドが警告0・エラー0で成功する。
