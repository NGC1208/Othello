# DDR-001：CPU難易度を3段階で提供する

- 状況: 採用
- 日付: 2026-06-18
- 決定者: プロジェクトオーナー

## 背景

当初のCPUはカジュアルプレイヤーに対して強すぎた。単純に合法手から無作為に選ばせると、弱いというより意図のない不自然な着手になり、対局体験を損なう。

また、強さを一種類に固定すると、初心者と経験者の双方へ適切な対局体験を提供できない。

## 企画上の目的

- 初心者から経験者まで、自分に合う強さを選べるようにする。
- 初級CPUには、目先の石数や辺を好む初心者らしい判断傾向を持たせる。
- 弱さを完全なランダム着手で表現しない。
- 難易度ごとの判断傾向をプレイヤーが理解できるようにする。

## 選択肢

1. CPUの強さを一種類に固定する。
2. 合法手から無作為に選ぶ難易度を追加する。
3. 最善手から一定確率で悪手を選ぶ。
4. 難易度ごとに判断傾向と先読み能力を変える。

## 決定

初級・中級・上級の3段階を提供する。

- 初級: 目先で多くの石を取る、初心者らしい判断を行う。
- 中級: 初級より先を読み、標準的な戦略を用いる。
- 上級: 中級より深く読み、現在提供する中で最も強い判断を行う。
- 完全なランダム着手は行わない。
- 同じ状況では一貫した判断を行う。
- ゲーム起動時と再対局時に難易度を選択する。
- 対局中は選択中の難易度を表示する。

具体的な評価式、探索方式、探索深度はADRで定義する。

## 受け入れ条件

- [x] 初級・中級・上級を対局開始前に選択できる。
- [x] 初級は目先の成果を好む初心者らしい手を選ぶ。
- [x] 中級と上級は初級より先を読んだ手を選ぶ。
- [x] 合法手から無作為に選択しない。
- [x] 同じ盤面と難易度では同じ手を選ぶ。
- [x] 再対局時に難易度を再選択できる。

## 影響

### 良い影響

- プレイヤーが自分の経験に合う難易度を選べる。
- 初心者が勝利を目指しやすくなる。
- 難易度ごとのCPUの個性を説明できる。

### 悪い影響・トレードオフ

- 難易度ごとのバランス調整とプレイテストが必要になる。
- 初心者らしさの評価は主観を含み、継続的な調整が必要になる。
- 難易度選択UIと難易度別テストが必要になる。
