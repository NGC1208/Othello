# ゲームルール

このファイルを対局ルールの正本とします。

## 盤面

- 盤面サイズ: 8 × 8
- プレイヤー数: 2
- 石の色: 黒・白
- 先手: 黒

## 初期状態

- 中央4マスに白と黒の石を互い違いに置く。

```text
盤面左上をA1とする。

D4: 白
E4: 黒
D5: 黒
E5: 白
```

## 合法手

- 石を置く場所は空きマスであること。
- 8方向のうち1方向以上で、隣接する1個以上の相手石に続いて自分の石があること。
- 途中に空きマスがある方向は成立しない。


## 石の反転

- 合法手が成立したすべての方向について、置いた石と自分の石に挟まれた相手の石をすべて反転する。

## パス

- 合法手がない場合は石を置かず、手番を相手に譲る。
- パス後、相手にも合法手がない場合は終局とする。

## 終局条件

- 空きマスがなくなった場合は終局とする。
- 空きマスが残っていても、両者とも合法手がない場合は終局とする。

## 勝敗

- 勝敗判定: 終局時に石の数が多い方を勝者とする。
- 同数の場合: 引き分けとする。

## 追加ルール

- 採用する追加ルール: なし
- 採用しない追加ルール: 標準ルール以外のすべて
